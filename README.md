### Local Development
> - Push all changes to remote repositories.
```bash
# Change permission for the git script.
chmod +x git-push-all

# Push all the change in remote repositories.
./git-push-all
```

> - Build and run the app.

```bash
# Change permission for the build script.
chmod +x build

# Run the app.
./app
```
